# Plan & Chill

Простий планувальник дня з  алгоритмом розстановки задач по помодоро.

---

## Структура проекту

```
plan-chill/
├── index.html           Сторінка входу (логін)
├── planner.html         Головна сторінка планувальника
├── settings.html        Налаштування акаунту
├── css/
│   └── global.css       Усі стилі
├── js/
│   └── utils.js         Спільна логіка, API, утиліти (ES модуль)
├── Backend.py           Python FastAPI бекенд
└── README.md
```

---

## Повний запуск

Відкрийте **два** термінали у папці проекту:

**Термінал 1 — Бекенд:**
```
pip install fastapi uvicorn

python Backend.py
```

**Термінал 2 — Фронтенд:**
```
python -m http.server 5173
```

Відкрийте браузер: **http://localhost:5173**

---

## Використання

1. **Вхід** — введіть ім'я, email та пароль (зберігаються локально у браузері).
2. **Планер** — оберіть день стрілочками або перейдіть у вигляд місяця.
3. **Додати задачу** — натисніть «+ Додати подію або рутину»:
   - **Рутина** — фіксований час (заняття, сон тощо).
   - **Подія** — можна задати вручну або обрати **Автопідбір** (алгоритм сам знайде вільний слот).
4. **Генерація розкладу** — після додавання завдань натисніть «Згенерувати розклад» — відобразиться таймлайн.
5. **Редагування** — щоб змінити список, натисніть «Змінити список завдань».
6. **Налаштування** — доступні через ліву панель.
