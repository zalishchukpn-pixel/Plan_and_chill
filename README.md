# Plan & Chill

Простий планувальник дня з  алгоритмом розстановки задач по помодоро.



### Структура проекту
```
plan-chill/
│
├── index.html              Сторінка входу / реєстрації
├── planner.html            Головна сторінка планувальника
├── settings.html           Налаштування акаунту
│
├── css/
│   └── global.css          Усі стилі проекту
│
├── js/
│   ├── utils.js            Спільні утиліти: API-функції, іконки,
│   │                       sidebar, хелпери часу, палітра кольорів
│   ├── index.js            Логіка входу та реєстрації
│   ├── planner.js          Логіка планувальника (день / місяць,
│   │                       форма задач, таймлайн, черга)
│   └── settings.js         Логіка налаштувань акаунту і помодоро
│
├── Backend.py              Python FastAPI сервер + SQLite
├── planner.db              База даних (створюється автоматично)
└── README.md
```



## Повний запуск

Відкрийте два термінали у папці проекту:

**Термінал 1 — Бекенд:**
```
pip install fastapi uvicorn

python Backend.py
```

**Термінал 2 — Фронтенд:**
```
python -m http.server 5173
```

Відкрийте в браузері: **http://localhost:5173**

### База даних

Файл `planner.db` створюється автоматично при першому запуску бекенду.

**Аби побачити табличку бази даних можна скачати розширення SQLite Viewer (воно доступне лише для VScode)**


### Використання

1. **Вхід** — введіть ім'я, email та пароль.
2. **Планер** — оберіть день стрілочками або перейдіть у вигляд місяця.
3. **Додати задачу** — натисніть ``+ Додати подію або рутину``:
   - **Рутина** — фіксований час.
   - **Подія** — можна задати вручну або обрати **Автопідбір** (алгоритм).
4. **Генерація розкладу** — після додавання завдань натисніть «Згенерувати розклад» — відобразиться згенерований розклад.
5. **Редагування** — щоб змінити список.
6. **Налаштування** — доступні через ліву панель.
